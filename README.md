Run, write, store and share recipes to run on your projects.

## Installation

Install the gem

    $ gem install kitchen_boy

Update the kitchen_boy recipes

    $ kitchen_boy update

## Usage

Run/Install kitchen recipes

    $ kitchen_boy install devise
    or
    $ kitchen_boy run devise
